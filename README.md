# Jarurat Care - Healthcare Support Web App

### Tech Stack
* Frontend: HTML5, Tailwind CSS
* Logic: JavaScript (ES6)

### AI / Automation Idea
* **Automated FAQ Assistant:** A logic-based chatbot designed to provide instant answers to volunteers, reducing the manual workload for NGO coordinators.

### NGO Use-Case
This app allows Jarurat Care to scale their volunteer onboarding process and ensure that common questions are handled immediately through automation.
